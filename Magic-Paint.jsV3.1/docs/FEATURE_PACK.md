# Magic Paint 機能拡張パック

AI機能に加えて、制作全体を底上げする機能をまとめて追加しました。
すべて `AppCore/features/` に独立モジュールとして実装され、
既存コアには手を入れず、`redraw()` / `canvasCoords()` などをラップして動作します。

---

## 追加ファイル

```text
AppCore/features/
├ edit_tools.js       ... 複製 / 反転 / z順序 / 整列・分布 / スポイト / パレット管理
├ viewport.js         ... グリッド / スナップ / ズーム・パン / オニオンスキン
├ export_media.js     ... 動画(WebM) / スプライトシート / 連番PNG 書き出し
├ command_palette.js  ... コマンドパレット(⌘K) / ショートカット一覧(?)
├ autosave.js         ... 自動保存 & クラッシュ復旧
└ feature_toolbar.js  ... 上記をトップバー/メニューに統合するUI
```

## 機能一覧

### 🎬 メディア書き出し（export_media.js）
既存のPNG/JS/HTML書き出しに加えて、アニメーションを動画で出力できます。
- **WebM動画** — `MediaRecorder` でキャンバスを1周録画
- **スプライトシート** — 全フレームを格子状に並べた1枚のPNG
- **連番PNG** — 各フレームを個別のPNGとして連続ダウンロード
- **現在フレームPNG** — 今の画面を1枚

トップバーの🎬ボタン、Fileメニュー、または⌘K→「メディア書き出し」から。

### 🧰 編集ツール（edit_tools.js）
- **複製**（⌘D）— 選択図形を16pxずらして複製。複数選択対応
- **反転** — 選択範囲の中心を軸に左右／上下反転
- **z順序** — 最前面／最背面（複数選択対応）
- **整列** — 左/右/上/下/水平中央/垂直中央（単体はキャンバス基準、複数は選択範囲基準）
- **分布** — 3個以上を水平／垂直に均等配置
- **スポイト**（I）— ネイティブ EyeDropper API 優先、無ければキャンバスクリックで色取得
- **カスタムパレット** — 現在色をパレットに保存（右クリックで削除、`localStorage` 永続化）

### 🔭 ビューポート（viewport.js）
- **グリッド**（G）— 背景に格子と中心線を表示
- **スナップ** — グリッドに図形描画を吸着
- **ズーム**（⌘+ / ⌘- / ⌘0、Ctrl+ホイール）— 25%〜400%、右下にズームバッジ
- **パン**（Spaceキー＋ドラッグ）— キャンバスをスクロール
- **オニオンスキン** — アニメーションの前後フレーム位置を薄く表示（前=赤 / 後=青）

### ⌘ コマンドパレット（command_palette.js）
- **⌘K** — 全コマンドを検索して実行（↑↓で移動、Enterで実行）
- **?** — キーボードショートカット一覧を表示

### 💾 自動保存 & 復旧（autosave.js）
- 20秒ごと + ページ離脱前に `localStorage` へスナップショット
- 次回起動時、前回の作業を検出したら「復元しますか？」を提案
- FileメニューでON/OFF切替（デフォルトON）
- 保存時、ステータスバーに「自動保存」バッジが一瞬光る

## キーボードショートカット（追加分）

| キー | 動作 |
|---|---|
| ⌘K | コマンドパレット |
| ? | ショートカット一覧 |
| ⌘D | 複製 |
| I | スポイト |
| G | グリッド切替 |
| ⌘ + / ⌘ - / ⌘ 0 | ズームイン / アウト / リセット |
| Ctrl + ホイール | ズーム |
| Space + ドラッグ | パン |

既存のショートカット（⌘Z/⌘C/⌘V/⌘S/⌘P/⌘G/⌘U、ツールキー V R C T P L E など）はそのままです。

## 設計方針

- **コア非改変** — `AppCore/features/` 内で完結。`redraw()` と `canvasCoords()` は
  元の関数を保持したままラップ（他モジュールが直接呼んでも壊れない）
- **依存の存在チェック** — すべての機能は呼び出し前に対象関数の有無を確認するため、
  一部モジュールが無くても致命的エラーにならない
- **永続化は localStorage のみ** — サーバー追加不要。グリッド設定・カスタムパレット・
  自動保存・自動保存ON/OFF をブラウザに保存
- **読み込み順** — すべて `app.js` の後に読み込むことで、コアのグローバルが揃った状態で初期化

---

## 動作要件

- WebM書き出しは `MediaRecorder` + `canvas.captureStream` 対応ブラウザが必要
  （非対応時はトーストで通知し、他の書き出しは利用可）
- スポイトのネイティブ版は `EyeDropper API` 対応ブラウザ（Chrome/Edge等）
  非対応時はキャンバスクリック方式に自動フォールバック
- サーバー側の追加は不要（Python/Flask には触れていません）
